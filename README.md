# Diabetes Bot

## Описание
Телеграм-бот для помощи диабетикам 2 типа:
- распознаёт еду по фото через GPT-4o
- рассчитывает углеводы и ХЕ
- (в будущем) подсказывает дозу инсулина
- ведёт простой дневник
